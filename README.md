# 国科大课程评价

## 这个 skill 是什么作用

`$ucas-course-evaluation` 用于全自动完成中国科学院大学/国科大选课系统里的课程评价和教师评价。

它会在已登录的评教页面中自动识别待评价项目，跳过已经完成的项目，为客观题选择最高评价，填写简洁积极的中文主观题，点击保存/提交，并检查评价状态是否已经完成。

适合的任务包括：

- 全自动完成当前学期的课程评价
- 全自动完成当前学期的教师评价

## 实现原理

这个 skill 通过可控浏览器执行评教流水线，优先使用浏览器控制能力，必要时再降级到桌面自动化。

执行优先级：

1. 浏览器控制技能：优先使用，例如 Chrome-control 或 in-app Browser。它可以读取页面结构、定位表单控件、设置单选项、填写文本框并点击保存按钮。
2. Computer Use：当浏览器控制技能不可用时使用，通过视觉识别、鼠标和键盘操作已打开的浏览器页面。
3. 无可控浏览器或桌面控制能力：无法执行网页自动化，只能报告缺少执行通道。

核心流程：

1. 接管或聚焦已登录的国科大评教页面。
2. 识别课程评价和教师评价入口。
3. 读取评价列表，区分待评价和已完成项目。
4. 打开待评价或待修改表单。
5. 自动选择每个客观题的最高评价。
6. 根据主观题提示填写中文评价。
7. 点击保存/提交。
8. 返回列表并检查项目是否变为已完成状态。
9. 重复处理，直到没有待评价项目。

默认中文主观题模板：

```text
课程内容充实，重点清晰，理论与应用结合较好，对理解相关知识帮助很大。
```

```text
暂无明显建议，整体教学安排合理，内容充实，学习收获较大。
```

```text
老师授课认真负责，讲解清晰，能够结合课程内容进行启发式讲授，课堂效果很好。
```

## 使用方法

先在浏览器中登录国科大选课系统，并打开课程评价或教师评价页面。

然后在 Codex 中使用：

```text
使用 $ucas-course-evaluation，全自动完成国科大课程评价和教师评价，都给最高评价，主观题用中文。
```

如果没有可用的浏览器控制技能，需要打开已登录的浏览器窗口，并允许 Codex 使用 Computer Use 进行桌面自动化。
