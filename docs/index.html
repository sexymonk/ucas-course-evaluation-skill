Input / Output Contract

从评教页面到完成状态

INPUT

已登录的评教页面

课程评价或教师评价入口、待评价列表、表单控件和主观题提示。

AGENT PIPELINE

浏览器优先的自动执行链

读取页面结构，定位待处理行，填选最高评价，写入中文回答，保存并复核状态。

OUTPUT

完成的课程与教师评价

已提交或已保存的评价项、跳过的已完成项，以及可核对的处理摘要。

Execution Model

三层控制策略

  1. 01

    Browser Control First

    优先通过可控浏览器读取 DOM、设置表单、点击保存，并用页面状态验证结果。

  2. 02

    Desktop Automation Fallback

    当浏览器控制不可用但页面可见时，使用桌面自动化完成同一套流程。

  3. 03

    Completion Check

    每次保存后回到列表或读取反馈，确认待评价项已经转为完成状态。

Prompt

使用方式

先在浏览器中登录国科大选课系统，并打开课程评价或教师评价页面，然后在 Codex 中调用 skill。

使用 $ucas-course-evaluation，全自动完成国科大课程评价和教师评价，都给最高评价，主观题用中文。
